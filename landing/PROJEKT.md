# Лендинг — Студия вокала «Твой Голос»

## Что это
Продающая страница студии вокала. Многосекционный лендинг с преимуществами, целевой аудиторией, пользой для здоровья и CTA на запись.

## Для кого
- Холодная аудитория из рекламы ВК
- Люди, которые ищут вокал в городе
- Те, кто сомневается и хочет узнать подробности

## Какие страницы уже есть
- `index.html` — лендинг (одна страница, всё в одном файле)

## Секции на странице
1. **Hero** — логотип, заголовок, слоган, CTA «Записаться на пробное»
2. **Преимущества** — 4 карточки: без нот, без абонементов, гибкое расписание, индивидуально
3. **Для кого** — 4 карточки: начинающие, мечтавшие, взрослые, дети
4. **Здоровье** — блок с пользой вокала для здоровья
5. **Нижний CTA** — повторный призыв к записи
6. **Футер** — копирайт + маркировка рекламы ФЗ-38

## Стиль дизайна
- Тёмный космический фон с warp-эффектом (canvas)
- Акцентный цвет: бирюзовый (#00d2c8) + фиолетовый (#a855f7) в градиентах
- Карточки с glass-эффектом
- Scroll-анимации (fade-up)
- Адаптивная вёрстка
- Без внешних библиотек

## Планируемые секции
- Отзывы учеников
- Секция «Как проходит занятие»
- Цены
- FAQ
