# Бот онлайн-записи — Telegram

## Что это
Универсальный Telegram-бот для онлайн-записи клиентов на услуги. Клиент выбирает услугу, дату, время — и записан. Мастеру приходит уведомление. За час до визита — напоминание.

## Для кого
- Мастера маникюра, барберы, массажисты, косметологи
- Любой специалист из сферы услуг, который ведёт запись

## Текущее состояние
В проекте есть 3 версии бота (дубли):
- `manicure-bot/` — бот для маникюра
- `vocal-bot/` — бот для записи на вокал
- `product/bot/` — шаблонный бот для продажи

Нужно объединить в один универсальный бот с config.json.

## Функции бота
### Для клиента:
- /start — приветствие + меню
- Записаться: услуга → дата → время → подтверждение
- Мои записи: просмотр + отмена

### Для мастера (админ):
- /today — записи на сегодня
- /tomorrow — записи на завтра
- /week — записи на неделю
- /stats — статистика за месяц
- Уведомления о новых записях и отменах

### Автоматика:
- Напоминание клиенту за 1 час до визита
- Занятые слоты не показываются

## Стек
- Python 3.11+, aiogram 3, SQLite (aiosqlite), APScheduler
- Конфигурация через config.json + .env

## Планируемые функции
- Рассылка клиентам от мастера
- Повторная запись в 1 клик
- Интеграция с Google Calendar
