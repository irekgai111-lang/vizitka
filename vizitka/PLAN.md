## 🎯 Рекомендации от маркетинговой команды (не реализовано, обсудить с Иреком)

### Приоритет 1 — сделать в первую очередь
- [ ] **Дубль CTA внизу каждого раскрытого блока-аккордеона.** Клиент читает про этапы голоса, мотивация пиковая — именно тут должна быть кнопка «Записаться», а не прокрутка вверх.
- [ ] **Уникальное обещание «до/после».** Добавить: «На пробном занятии делаем запись до и после — слышно разницу своими ушами». Это выделяет из конкурентов, которые все обещают «бесплатный пробный».
- [ ] **Блок «Для кого это».** 3 короткие строчки на первом экране: «для взрослых, кто давно хочет петь / для свадеб-юбилеев / онлайн из любого города». Сегментация = клиент узнаёт себя.
- [ ] **Фото Ирека** (не логотип). Живое фото → доверие ×3. Пришлёт файл — вставить рядом с логотипом или вместо.
- [ ] **Реальная цена и 2-3 тарифа.** Сейчас «от 1500 ₽» вслепую. Лучше: 3 тарифа (пробный 0 ₽ / абонемент 8 уроков / индивидуальный курс до сцены) — средний продаётся лучше всего.

### Приоритет 2 — докрутить перформанс
- [ ] **UTM-метки на CTA-кнопках.** `?utm_source=qr&utm_medium=offline_card` — чтобы понимать, откуда пришёл клиент (бумажная визитка vs онлайн-шаринг).
- [ ] **Отдельный Telegram для школы.** Сейчас CTA ведёт на `@irek_veduschii` (ведущий мероприятий). Нужен `@tvoigolos_nk` или аналог — отдельный канал/бот именно для школы вокала.
- [ ] **Кнопка «Все отзывы на 2ГИС» — переосмыслить.** Сейчас она выводит клиента с визитки (теряем лид). Варианты: (а) показать только после клика по CTA, (б) убрать совсем, (в) открывать в новом окне и «отпускать» только прогретого лида.
- [ ] **Лид-магнит для холодных.** Не все готовы сразу на пробный. Нужна «лестница»: «3 упражнения для голоса — бесплатно в Telegram» → прогрев → пробное.
- [ ] **Мини-срочность.** «Беру на индивидуальные занятия максимум 6 человек. Сейчас свободно 2 слота» — честно и работает.

### Приоритет 3 — бренд-работа
- [ ] **Определиться: Ирек vs «Твой Голос».** Сейчас два бренда на одной странице конкурируют. Решить — это визитка личного педагога или школы. От этого зависит позиционирование на будущем лендинге.
- [ ] **Единый tone of voice.** Сейчас микс «ты»/«вы». Рекомендую «на ты», тёплый и прямой — вокал это эмоция, «вы» дистанцирует.
- [ ] **Обещание трансформации, а не процесса.** Заменить формулировки про «чистую интонацию», «тембр» на сцены из жизни клиента: «спеть маме на юбилей», «петь в караоке без стыда», «подарить песню жене».
- [ ] **Звуковой/видеоэлемент.** Визитка про вокал должна звучать. Короткое аудио «голос ученика до → после» (5-10 секунд) на главной. Если не аудио — то видео.
- [ ] **Мета-теги + Open Graph** (description, og:image) — чтобы при шаринге ссылки в мессенджерах показывалась красивая превью-карточка.
- [ ] **Favicon** — иконка в табе браузера.

### Приоритет 4 — долгосрочное
- [ ] **Отдельный лендинг/визитка для «Ведущего мероприятий».** Своя ЦА, свой оффер, свой QR. Не смешивать с вокалом.
- [ ] **Воронка-квиз** после CTA (вместо Telegram-диалога): «Сколько вам лет? / Пели ли раньше? / Ваша цель?» → бот доводит до записи и греет рассылкой.
- [ ] **Метрика/Pixel** для ретаргетинга.
- [ ] **UGC и хештег школы** для сбора отзывов учеников в соцсетях.
- [ ] **Видео учеников «до/после»** — 3-5 коротких на главной, автоплей с mute.
