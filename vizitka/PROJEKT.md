# Визитка — «Твой Голос»

## Что это
Онлайн-визитка преподавателя вокала Ирека Гайнемухаметова. Одностраничный сайт-карточка с контактами, описанием методики и ссылками на соцсети.

## Для кого
- Потенциальные ученики, которые увидели ссылку в соцсетях или на визитке
- Люди, которым скинули ссылку по рекомендации

## Какие страницы уже есть
- `index.html` — визитка (одна страница, всё в одном файле)

## Что на странице
- Логотип «Твой Голос» (SVG-микрофон, анимированный)
- Имя, специализация, слоган
- 3 раскрывающихся блока: Методика, Результат, Обо мне
- Ссылка на отзывы (ВК)
- CTA-кнопка «Написать мне» (Telegram)
- Соцсети: Telegram, VK, WhatsApp, Max, Телефон

## Стиль дизайна
- Тёмный космический фон (#0a0a0a) с анимированными звёздами
- Акцентный цвет: бирюзовый (#00d2c8)
- Карточка по центру с glass-эффектом
- Анимации: пульсация, вращение колец, мерцание
- Адаптивная вёрстка (мобильная версия)
- Без внешних библиотек — всё в одном HTML

## Планируемые страницы
- Пока одна страница. Возможно: добавить секцию с ценами или расписанием
