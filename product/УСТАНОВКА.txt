========================================
  BOT BUSINESS — Быстрый старт
========================================

ШАГ 1: Установи Python
  — Скачай с https://python.org
  — При установке ОБЯЗАТЕЛЬНО поставь галочку "Add to PATH"

ШАГ 2: Создай бота в Telegram
  — Открой Telegram, найди @BotFather
  — Напиши /newbot
  — Придумай название и username
  — Скопируй токен

ШАГ 3: Узнай свой Telegram ID
  — Найди @userinfobot в Telegram
  — Напиши /start
  — Скопируй число (это твой ID)

ШАГ 4: Настрой бота
  — Открой папку bot\
  — Скопируй файл .env.example → переименуй в .env
  — Открой .env блокнотом
  — Вставь токен и ID

ШАГ 5: Запусти
  — Дважды кликни ЗАПУСК.bat
  — Готово! Бот работает.

========================================
  Как адаптировать под клиента
========================================

Открой файл bot\database.py в блокноте.
Найди раздел SERVICES — там услуги, цены, длительность.
Меняй под любую нишу:

  Маникюр, барбер, массаж, репетитор, фотограф, автомойка...

========================================
  Полное руководство по бизнесу
========================================

Открой файл БРОШЮРА.md — там всё:
  — Скрипт продаж
  — Где искать клиентов
  — Сколько зарабатывать
  — Работа с возражениями
  — Чеклист запуска
