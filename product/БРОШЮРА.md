## ШАГ 3. Создай демо-бот

### Что такое Claude Code (простыми словами)

Claude Code — это программа, которой ты пишешь на русском языке что тебе нужно, а она пишет за тебя код. Ты не программист — и не надо им быть. Ты просто описываешь задачу, Claude Code создаёт файлы.

### Что делаешь

1. @BotFather в Telegram → `/newbot` → получи токен
2. @userinfobot → `/start` → получи свой ID
3. Открой Claude Code и вставь техзадание ниже
4. В `.env` вставь токен и ID
5. `pip install -r requirements.txt && python bot.py`

### Техзадание для Claude Code — скопируй и вставь

```
Создай Telegram-бота для онлайн-записи клиентов на услуги.

СТЕК: Python 3.11+, aiogram 3, SQLite (aiosqlite), APScheduler.

СТРУКТУРА ПРОЕКТА:
bot/
├── bot.py           — точка входа, запуск бота
├── config.py        — загрузка переменных из .env
├── database.py      — работа с БД (создание таблиц, CRUD)
├── handlers/
│   ├── client.py    — хендлеры для клиента (запись, просмотр, отмена)
│   └── admin.py     — хендлеры для мастера (расписание, управление)
├── keyboards.py     — все inline-клавиатуры
├── scheduler.py     — напоминания клиентам
├── .env             — токен и ID (не коммитить)
└── requirements.txt — зависимости

ФАЙЛ .env:
BOT_TOKEN=токен_от_botfather
ADMIN_ID=telegram_id_мастера

ФУНКЦИИ КЛИЕНТА (любой пользователь):
1. /start — приветствие + главное меню (inline-кнопки)
2. «Записаться» — выбор услуги → выбор даты (ближайшие 14 дней) → выбор свободного времени → подтверждение → запись сохранена
3. «Мои записи» — список активных записей с кнопкой «Отменить»
4. При отмене — мастеру приходит уведомление

ФУНКЦИИ МАСТЕРА (только ADMIN_ID):
1. /today — записи на сегодня (время, клиент, услуга)
2. /tomorrow — записи на завтра
3. /week — записи на неделю
4. /stats — статистика за месяц (сколько записей, сколько новых клиентов, сколько отмен)
5. При новой записи — мастеру приходит сообщение: «Новая запись: [услуга], [дата], [время], клиент: @username»
6. При отмене — мастеру приходит сообщение

УСЛУГИ ДЛЯ ДЕМО:
- Маникюр: 90 мин, 1 500 ₽
- Педикюр: 60 мин, 1 200 ₽
- Маникюр + педикюр: 120 мин, 2 500 ₽
- Покрытие гель-лак: 45 мин, 800 ₽

РАСПИСАНИЕ: Пн-Сб, 09:00–20:00. Вс — выходной.
Слоты генерируются автоматически с учётом длительности услуги.
Занятые слоты не показываются.

НАПОМИНАНИЯ:
- За 1 час до визита — сообщение клиенту: «Напоминаем: [услуга] сегодня в [время]»
- Проверка каждые 5 минут через APScheduler

ТРЕБОВАНИЯ:
- Асинхронный код (async/await)
- Все тексты на русском
- Inline-клавиатуры (не reply)
- Callback-запросы с answer_callback_query
- Корректная обработка часовых поясов (МСК)
- Бот не падает при ошибках — try/except с логированием
- requirements.txt с фиксированными версиями
```

**Результат:** работающий бот. Открой, нажми Start, пройди весь путь записи.

### Если что-то пошло не так (шаг 3)

| Что видишь | Что делать |
|------------|-----------|
| `python не является внутренней командой` | Python не установлен или нет галочки "Add to PATH". Переустанови Python, поставь галочку |
| `ModuleNotFoundError: No module named 'aiogram'` | Ты не установил зависимости. Выполни `pip install -r requirements.txt` |
| `Invalid token` | Токен в `.env` неправильный. Скопируй заново из @BotFather |
| `Chat not found` по ADMIN_ID | ID в `.env` неправильный. Проверь через @userinfobot |
| Бот запустился, но не отвечает | Напиши боту `/start`. Если молчит — скопируй текст ошибки из терминала и отправь в Claude Code: «Бот не отвечает, вот ошибка: [вставь ошибку]» |
| Красный текст с `Error` или `Traceback` | Скопируй ВСЁ что написано красным, вставь в Claude Code и напиши: «Исправь эту ошибку» |

> **Золотое правило:** если видишь ошибку — скопируй её целиком и отправь в Claude Code со словами «Исправь». В 90% случаев он починит сам.

## ШАГ 5. Арендуй сервер

### Зачем это СЕЙЧАС, а не позже

Пока бот на твоём компьютере — он умирает когда ты закрываешь ноутбук. Нельзя идти продавать, пока бот не работает 24/7. Поэтому сервер — ДО продаж.

### Что такое VPS (простыми словами)

VPS — это компьютер в интернете, который никогда не выключается. Ты платишь 150 ₽/мес — и у тебя есть компьютер, который работает 24/7. На нём живут твои боты.

### Что делать

1. Зайди на **timeweb.cloud**
2. Зарегистрируйся
3. Создай сервер: Ubuntu, минимальный тариф (~150 ₽/мес)
4. Получишь IP-адрес и пароль

### Техзадание для Claude Code — скопируй и вставь

```
Помоги мне настроить VPS сервер для запуска Telegram-ботов.

Я арендовал VPS на Timeweb. Данные:
- IP: [вставь свой IP]
- Пароль: [вставь свой пароль]
- ОС: Ubuntu

Я НЕ программист. Объясняй каждый шаг простыми словами.
Говори что вводить и куда (какую программу открыть на моём компьютере).

Нужно пошагово:
1. Подключиться к серверу с моего компьютера (Windows)
2. Установить Python на сервере
3. Загрузить файлы моего бота на сервер
4. Запустить бот так, чтобы он работал всегда — даже после перезагрузки
5. Показать команды: как проверить работает ли бот, как перезапустить, как посмотреть ошибки
6. Показать как добавить второй бот на тот же сервер
```

### Если что-то пошло не так (шаг 5)

| Что видишь | Что делать |
|------------|-----------|
| Не могу подключиться к серверу | Проверь IP и пароль. Попробуй через 5 минут (сервер может ещё создаваться) |
| `Connection refused` | Сервер не запущен. Зайди в панель Timeweb → проверь статус |
| Бот работал, потом перестал | Подключись к серверу, набери: `sudo systemctl status bot1`. Скопируй вывод в Claude Code |
| Не понимаю что вводить и куда | Скажи Claude Code: «Я на Windows. Объясни как открыть терминал и подключиться к серверу пошагово, как для полного новичка» |

### Что входит в подписку (чтобы клиент не отписался)

| Что | Зачем |

## ШАГ 9. Telegram-канал для прогрева

**Почему Telegram:** охват поста 50–70% подписчиков (ВК — 10–20%), бот уже там, вести проще.

**Оформление:**
- Название: «Онлайн-запись для мастеров»
- Описание: ссылка на демо-бот + твой контакт
- Аватарка: простая, с иконкой календаря

**Контент — 3 поста в неделю:**

| Тип | Пример | Зачем |
|-----|--------|-------|
| Кейс | «Настроил бот для Анны, за неделю 23 записи» | Доказательство |
| Боль | «Мастер работает, 3 клиента пишут — не может ответить» | Узнаваемость проблемы |
| Демо | Видео-кружок: запись за 20 секунд | Визуальное доказательство |
| Отзыв | Скрин от клиента | Социальное доказательство |

**Как канал работает в воронке:**

```
Ты написал мастеру → она не ответила, но подписалась
    ↓
Через неделю видит кейс: «У Маши 30 записей через бот»
    ↓
Пробует демо-бот из описания канала
    ↓
Пишет тебе сама: «Сколько стоит?»
```

Канал — твой молчаливый продавец. Работает даже когда ты спишь.
