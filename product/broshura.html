3 Создай демо-бот

Что такое Claude Code

Claude Code — это программа, которой ты пишешь на русском языке что тебе нужно, а она пишет за тебя код. Ты не программист — и не надо им быть.

Что делаешь

  1. @BotFather в Telegram → /newbot → получи токен
  2. @userinfobot → /start → получи свой ID
  3. Открой Claude Code и вставь техзадание ниже
  4. В .env вставь токен и ID
  5. pip install -r requirements.txt && python bot.py

Техзадание для Claude Code — скопируй и вставь

Создай Telegram-бота для онлайн-записи клиентов на услуги.

СТЕК: Python 3.11+, aiogram 3, SQLite (aiosqlite), APScheduler.

СТРУКТУРА ПРОЕКТА:
bot/
├── bot.py           — точка входа, запуск бота
├── config.py        — загрузка переменных из .env
├── database.py      — работа с БД (создание таблиц, CRUD)
├── handlers/
│   ├── client.py    — хендлеры для клиента
│   └── admin.py     — хендлеры для мастера
├── keyboards.py     — все inline-клавиатуры
├── scheduler.py     — напоминания клиентам
├── .env             — токен и ID (не коммитить)
└── requirements.txt — зависимости

ФАЙЛ .env:
BOT_TOKEN=токен_от_botfather
ADMIN_ID=telegram_id_мастера

ФУНКЦИИ КЛИЕНТА:
1. /start — приветствие + главное меню (inline-кнопки)
2. «Записаться» — выбор услуги → дата → время → подтверждение
3. «Мои записи» — список активных с кнопкой «Отменить»
4. При отмене — мастеру приходит уведомление

ФУНКЦИИ МАСТЕРА (только ADMIN_ID):
1. /today — записи на сегодня
2. /tomorrow — записи на завтра
3. /week — записи на неделю
4. /stats — статистика за месяц
5. При новой записи — уведомление мастеру
6. При отмене — уведомление мастеру

УСЛУГИ ДЛЯ ДЕМО:
- Маникюр: 90 мин, 1 500 ₽
- Педикюр: 60 мин, 1 200 ₽
- Маникюр + педикюр: 120 мин, 2 500 ₽
- Покрытие гель-лак: 45 мин, 800 ₽

РАСПИСАНИЕ: Пн-Сб, 09:00–20:00. Вс — выходной.
Слоты автоматически с учётом длительности.

НАПОМИНАНИЯ: За 1 час до визита через APScheduler.

ТРЕБОВАНИЯ:
- Асинхронный код (async/await)
- Все тексты на русском
- Inline-клавиатуры
- Обработка часовых поясов (МСК)
- try/except с логированием
- requirements.txt с фиксированными версиями

Результат: работающий бот. Открой, нажми Start, пройди весь путь записи.

4 Сделай бот универсальным

Техзадание для Claude Code

Рефакторинг бота: вынеси все настройки клиента в config.json.

Создай файл config.json:
{
  "business_name": "Студия маникюра Анны",
  "services": [
    {"id": "manicure", "name": "Маникюр", "emoji": "💅",
     "duration": 90, "price": 1500},
    {"id": "pedicure", "name": "Педикюр", "emoji": "🦶",
     "duration": 60, "price": 1200}
  ],
  "schedule": {

Валидация: если config.json битый — понятная ошибка.

Результат: новый клиент = заполнить 1 файл за 5 минут.

5 Арендуй сервер

Зачем это СЕЙЧАС

Пока бот на твоём компьютере — он умирает когда ты закрываешь ноутбук. Нельзя продавать, пока бот не работает 24/7.

Что такое VPS

VPS — это компьютер в интернете, который никогда не выключается. 150 ₽/мес — и у тебя есть машина 24/7. На ней живут все боты.

Что делать

  1. Зайди на timeweb.cloud
  2. Зарегистрируйся
  3. Создай сервер: Ubuntu, минимальный тариф (~150 ₽/мес)
  4. Получишь IP-адрес и пароль

Техзадание для Claude Code

Данные:
- IP: [вставь свой IP]

Я НЕ программист. Объясняй каждый шаг простыми словами.

Нужно пошагово:
1. Подключиться к серверу с Windows
2. Установить Python
3. Загрузить файлы бота
4. Запустить бот чтобы работал всегда (даже после перезагрузки)

6 Найди первый трафик

Ниша: маникюр. Город: твой. Не распыляйся.

Канал 1 — Ближний круг (день 1)

  • Твой мастер маникюра
  • Мастера знакомых: «У тебя есть мастер маникюра? Можешь познакомить?»

Канал 2 — Instagram/ВК (день 2–5)

  1. Instagram → поиск → маникюр [город]
  2. Ищи тех, у кого «запись в личку» — это твой клиент
  3. ВК → поиск по сообществам → маникюр [город]
  4. 2ГИС → «маникюр» → список с контактами

Темп: 10–15 сообщений в день.

Канал 3 — Ноги (день 3–5)

Ближайший ТЦ. Салоны на каждом этаже. Заходи, показывай бот на телефоне.

Канал 4 — Авито (параллельно)

Онлайн-запись для мастеров маникюра. Telegram-бот: клиенты записываются сами, напоминание за час. Покажу демо за 20 секунд.

Реальная конверсия (честно)

  • Из 10 сообщений: 2–3 ответят
  • Из 3 ответов: 1 посмотрит демо
  • Из 3 демо: 1 купит
  • Итого: 30–50 сообщений = 1 клиент

Не расстраивайся от игноров — это нормально. 90% не ответят. Тебе нужен 1 из 10.

Чеклист шага 6

  • Отправил 30+ сообщений мастерам маникюра
  • Обошёл минимум 1 ТЦ
  • Объявление на Авито
