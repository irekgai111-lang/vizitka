# BOT BUSINESS — Готовый бизнес по продаже ботов онлайн-записи

## Что это?
Полный пакет для запуска бизнеса по продаже Telegram-ботов для бьюти-мастеров и сферы услуг.

## Что внутри
- Готовый бот для записи клиентов (Python + aiogram 3 + SQLite)
- Брошюра-руководство по запуску бизнеса (PDF-ready)
- Скрипт продаж
- Шаблоны сообщений для поиска клиентов

## Быстрый старт
1. Прочитай `БРОШЮРА.md` — там всё от А до Я
2. Настрой демо-бота по инструкции
3. Найди первого клиента по скрипту
4. Заработай первые деньги
