# Продукт — Bot Business (Брошюра)

## Что это
Готовый бизнес-пакет для заработка на установке ботов онлайн-записи мастерам. Включает пошаговую брошюру-инструкцию и шаблон бота.

## Для кого
- Люди, которые хотят начать зарабатывать на ботах
- Не-программисты — всё делается через Claude Code

## Какие файлы уже есть
- `БРОШЮРА.md` — пошаговая инструкция (10 шагов: от создания бота до автономного бизнеса)
- `БРОШЮРА.html` — HTML-версия (старая, нужно обновить)
- `bot/` — шаблон бота для продажи (базовая версия)
- `ЗАПУСК.bat` — батник для запуска
- `УСТАНОВКА.txt` — инструкция по установке
- `README.md` — описание

## Что в брошюре
1. Понимание продукта + УТП vs YCLIENTS
2. Самозанятость + договор
3. Создание бота (ТЗ для Claude Code)
4. Универсальный config
5. Сервер 24/7 (ТЗ для Claude Code)
6. Поиск трафика
7. Скрипт продаж (3 техники)
8. Тарифы и закрытие сделки
9. Telegram-канал для прогрева
10. Автономная система

## Стиль
- Простой язык, как инструкция к мебели
- Чеклисты после каждого шага
- Готовые шаблоны: договор, отчёт, сообщения
