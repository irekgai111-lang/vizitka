# CLAUDE.md — правила работы с проектом «Онлайн вокал студия»

## Контекст, которого всегда придерживаемся

1. **Имя проекта:** «Онлайн вокал студия «Твой Голос»», Нижнекамск + онлайн.
2. **Владелец:** Ирек — обращение на «ты», по-русски, без канцелярита.
3. **Главное узкое место:** конверсия пробное → оплата (13%). Любое действие оцениваем через призму «поднимет это конверсию или нет?».
4. **Позиционирование:** эксперт с опытом 30 лет + 300+ учеников. Не повторять стиль блогеров («ВАУ-акция!!!», эмодзи-салюты, каплок).

## Правила для любого ответа по проекту

- **Сначала бизнес-эффект, потом креатив.** Не предлагать инструмент без связки с воронкой.
- **Юрида обязательна.** ФЗ-38 (реклама), ФЗ-152 (ПД), Meta/Instagram запрещены в РФ — не писать «Instagram», не давать ссылок.
- **Не перегружать.** 1 действие в день = лучше, чем 10 в план. Ирек работает один + преподаёт.
- **Цифры — только проверяемые.** 300+ учеников, ★5.0 на 2ГИС, 41 отзыв, 296 отзывов в ВК — только так. Не суммировать 41+296.
- **Сменники (50% базы)** — всегда учитывать: плавающий график, оплата частями, смены Нефтехима.
- **Фото и видео учеников** — только с их согласия (у Ирека есть сменные работники, им публичность может быть нежелательна).
- **«Более 300»** — округление вниз, допустимо. «Более 400» или «лучшая школа» — нет, ФАС оштрафует.

## Приоритет задач

1. Всё, что повышает конверсию пробного (скрипт, антивозражения, пост-касания) — **высший приоритет**.
2. Контент для прогрева (ВК, TG) — **средний**.
3. Новые каналы (онлайн-курс, таргет, квиз) — **низкий**, включаем только после фиксации пункта 1.

- Примеры живые, из Нижнекамска (Нефтехим, Сююмбике 7, сменщики).

## Чего не делаем

- Не запускаем таргет пока конверсия < 25%.
- Не делаем квиз пока не отработан скрипт пробного.
- Не мешаем личный Instagram (irekgai111) и школьный бренд — они остаются раздельно.
